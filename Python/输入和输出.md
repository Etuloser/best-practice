# 输入和输出

###输出

```python
>>> print('hello, world')
```

####多字符串输出

> 1. 用逗号隔开
> 2. print() 方法遇到逗号会打印一个空格

```python
>>> print('The quick brown fox','jumps over','the lazy dog')
The quick brown fox jumps over the lazy dog
```

#### 打印整数，或者计算结果

```python
>>> print(300)
300
>>> print(100 + 200)
300
```
